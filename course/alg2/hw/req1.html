Homework 1
( Strongly Connected Components )

Recall that a Strongly Connected Component of a directed graph G = (V,E) is a maximal set of vertices C included in V such that for every pair of vertices u and v in C, we have both u->v and v->u.

Implement the algorithm in ANSI C only. (no C++)
The prototype of your function is:

void strongly_connect_component( int number_of_vertex);

in hw1.c. You can try to run this file first.
You should complete the function and be saved as "ID.cpp" in one file, i.e. if your ID is b92902001, then your file should be b92922001.cpp
TA will examine your code so you have to show the answers ( The Strongly Connected Components of input graph)

The representation of directed graph is   adjacency-list (no adjacency-matrix).
The graph input information is in input.txt with the following format:

input.txt
-----------------
5
1 2
1 3
2 3
1 4
3 5

-----------------
First line show the vertex numbers. In other lines, first number shows the start-vertex and the second shows the end-vertex
