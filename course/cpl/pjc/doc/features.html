Type Definition

　　這裡的 Type difinition 是以 alias 的概唸完成的，雖然老師好像說要以 name equivalence 的方式完成，但是一旦使用 name equivalence 的方式 implement，則所有 type define 過的變數都只能跟同類型的變數比較，這樣等於不能 input 也不能 output，連 assign 值都不行。而 alias 的話，只是定義一個新名稱而已，本質還是那幾種 type。
