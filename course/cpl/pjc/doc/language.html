Intro

本程式為 PASCAL-like 的語言，提供了 INTEGER、REAL 兩種 elementary type，以及 multi-dimension array of elementary type，可任意設定 array 的 lower bound 跟 upper bound。 也提供了 type definition 跟 const declaration 的功能。提供以下 expression：

Arithmetic operator： (皆為 left association)

Operator Usage Comment
+ Add Scale value only
- Subtract Scale value only
* Multiply Scale value only
/ Divide OScale value only
MOD Modulo INTEGER value only
:= Assign Scale value only
<－> Swap Both scale value or aggregation type

Boolean operator：

Priority(from high to low)
Expression Association
NOT right
AND left
XOR left
OR left

Comparison operator:

Operator
>
<
=
⇐
>=
<>

本程式並提供了幾種基本的 IO：

Function Usage Comment
READ(non-empty-list-ofvariables) READ sequence of variables INTEGER and REAL only
WRITE(non-empty-list-ofvariables/constants) WRITE sequence of variables INTEGER and REAL only
WRITESP() output a single space
WRITELN() output a new line
WRITESTRING(c-string) output string

Syntax

PROGRAM program_name
CONST REGION (option)
TYPE REGION (option)
VAR REGION (option)
PROCEDURE/FUNCTION REGION(option and multiple)
BLOCK-REGION

以下將針對各部分加以說明

CONST REGION

其語法為： 

CONST
const_name = (const value | previous defined const_name);
...
ENDCONST

本程式僅提供 INTEGER 及 REAL 兩種 const value。以下為 example： 

CONST
pi = 3.14;
num = 10;
size = num;
ENDCONST

TYPE REGION

TYPE
type_name = (default type | previous defined type name);
...
ENDTYPE

本程式提供對於 INTEGER、REAL、array 做 type definition。以下為 example： 

TYPE
im2darray = ARRAY [-1..5,3..10] OF INTEGER;
num = INTEGER;
size = num;
ENDTYPE

VAR REGION

VAR
non-empty-list-of-names : type;
...
ENDVAR

每一行都會將 list 中的每一個 name 設定為 type 型態的變數。 

VAR
x,y,z : INTEGER;
a,b : REAL;
k : ARRAY [1..10,-10..-1] OF REAL;
ENDVAR

PROCEDURE/FUNCTION REGION

PROCEDURE REGION: 

PROCEDURE name (non-empty-list-of-names:type ; ... );
CONST REGION
TYPE REGION
VAR REGION
PROCEDURE/FUNCTION REGION
BLOCK REGION

FUNCTION REGION: 

FUNCTION name (non-empty-list-of-names:type ; ... ) : return_type ;
CONST REGION
TYPE REGION
VAR REGION
PROCEDURE/FUNCTION REGION
BLOCK REGION

此程式 PROCEDURE/FUNCTION 為 nested 定義，每個 PROCEDURE/FUNCTION 皆只能 call 出現在他之前的 PROCEDURE/FUNCTION。而要變更 FUNCTION 的 return value，需要 FUNCTION name 為 l-value 來存取。以下為 example： 

FUNCTION addOne(n) : INTEGER;
BEGIN
   addOne := n + 1;
END

BLOCK REGION

BEGIN
VAR REGION (option)
statement
...
END

BLOCK 內變數的生命週期僅存在於 BEGIN .. END 之內。

Statement

Condition

IF

IF bool-expression
THEN
   statement/block statement
ENDIF

IF..ELSE

IF bool-expression
THEN
   statement/block statement
ELSE
   statement/block statement
ENDIF

FOR..TO

FOR var := int-expression-1 TO int-expression-2
   statement/block statement

FOR..DOWNTO

FOR var := int-expression-1 DOWNTO int-expression-2
   statement/block statement

WHILE

WHILE( bool-expression ) 

 statement/block statement

CASE

CASE expression OF
   constant_1 : statement/block statement
   constant_2 : statement/block statement
   ...
   OTHERWISE :  statement/block statement
ENDCASE

Example

PROGRAM main
  FUNCTION fab(x : INTEGER): INTEGER;
    BEGIN
    	IF (x = 0) OR (x = 1)
	THEN
	   fab := 1;
	ELSE
	   BEGIN
	     fab := fab(x-1) + fab(x-2);
    	   END
	ENDIF;
    END
  
  BEGIN
    BEGIN
      VAR
        w, x, z: INTEGER;
      ENDVAR
      READ(w);
      FOR z := 0 TO w DO
      BEGIN
      	x := fab(z);
      	WRITE(x);
	WRITESP();
      END
      WRITELN();
    END
  END

End
