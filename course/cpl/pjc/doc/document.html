Project

Structure

  • .
    • Makefile
    • bin/
    • doc/
      • Language # Language reference manual
      • Features # List of features implemented and their corresponding test programs
      • Internal #Implementation manual contains the implementation details.
      • Other helpful documents: otherX.xxx
      • Can merge everything into one document with clearly marked sections of the above. Call this file document.xxx
    • lib/
      • cmm.c
    • src/
      • Makefile
      • emit.cpp # all emit code are here
      • emit.h
      • inc.h * configure file
      • library.cpp # library functions
      • library.h
      • OneSymbolTableByAVLTree.cpp # One symbol table implemented by AVL tree
      • OneSymbolTableByAVLTree.h
      • pcompiler
      • pparser.y # parser
      • pscaner.l # scaner
      • SymbolTable.h # General class of symbol table
    • tests/
      • programX.p: program.
      • inputX Y: input test data.
      • outputX Y: output data.
      • readmeX: documentation for programX, contains the purpose of having test programX.
      • # Example: program1.p, input1 1, input1 2, output1 1, output1 2 and readme1.

Installation

Just unpack it and type 『make' 

$ tar xzvf B92902106_proj.tar.gz
$ make

Usage

$ pcompiler file.p     # generates an executable object file "p.out"
$ p.out                # execute the compiled program "p.out"
$ pcompiler -a file.p  # generates a C– code file named 「file.out」
